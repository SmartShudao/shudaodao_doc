## ShuDaodao 上传文件 接口

主要通过 **upload api** 实现

- 适用于智能体的问答或文本生成任务。
- 适用于公司产品化，方便Agent的调用

### **1. 接口基本信息**

- **请求方法 API地址**:


| 请求方式 | API地址                       | 参数说明 |
|:----:|:----------------------------|:-----|
| POST | /api/v1/files/upload/{user} | 用户标识 |
| GET  | /api/v1/files/{user}        | 用户标识 |


- **认证方式**:
    - 自定义
